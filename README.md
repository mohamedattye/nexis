# Suivi logistique des camions (Nexis)

Plateforme web interne pour piloter vos opérations logistiques camion par camion.

## Ce qui a été amélioré

- Interface plus intuitive et plus visuelle
- Design modernisé pour un usage interne quotidien
- Intégration correcte du logo Nexis (entête + favicon navigateur)
- Graphiques de performance:
  - Revenus vs Dépenses (mois filtré)
  - Résultat net par camion (revue mensuelle)

## Utilisation

1. Ouvrir `index.html` dans votre navigateur.
2. Ajouter les courses (camion, date, zone, revenu, dépenses).
3. Filtrer par mois/camion.
4. Lire les cartes + graphiques + revue mensuelle par camion.
5. Exporter les courses filtrées au format CSV.

## Données

Les données sont sauvegardées dans le navigateur (`localStorage`).
